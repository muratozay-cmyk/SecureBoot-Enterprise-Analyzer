# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately.

Do not publish vulnerabilities publicly before they are reviewed.

Contact:

Murat Ozay

GitHub Issues (Security)

or

LinkedIn

This project does not transmit credentials.

Remote authentication uses Windows PowerShell Get-Credential only.

Credentials are stored only in the current PowerShell session memory.

No passwords are written into reports.
