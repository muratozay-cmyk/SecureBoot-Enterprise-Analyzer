# Frequently Asked Questions

## Does this tool modify Secure Boot?

No.

The tool is READ ONLY.

---

## Does this tool update certificates?

No.

---

## Does this tool enable Secure Boot?

No.

---

## Are passwords stored?

No.

Windows Get-Credential is used.

Passwords remain only in PowerShell session memory.

Nothing is written into reports.

Yes.
