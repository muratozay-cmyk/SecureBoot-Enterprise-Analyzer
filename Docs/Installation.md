# Installation

## Requirements

Windows PowerShell 5.1 or newer

Administrator privileges

Windows Server 2016+

Windows Server 2019

Windows Server 2022

Windows Server 2025

PowerShell Remoting (for remote assessment)

## Usage

Run

SecureBootEnterpriseAnalyzer.ps1

or

MainForm.ps1

Administrator privileges are recommended.

Reports will automatically be generated under:

Desktop\Reports
